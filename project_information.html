Online Course Website Development

SIT, August 2022 - December 2022

During this project, I developed an online course website using Node.js technologies. Some of the features I implemented include:

  • A contact form
  • A search bar
  • Options for creating a new course
  • Access to enroll/un-enroll
  • Edit profiles
  • Delete courses options for both students and instructors

To improve website performance and enhance user experience, I integrated APIs and utilized the axios library.
